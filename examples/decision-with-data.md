# Decision With Data Example

## Situation

You need a recommendation for a pricing-page experiment using Slack context, one ticket thread, and a Hex chart export.

## Input

```text
Use decision-brief.
Pull in the experiment thread, the launch ticket, and the attached Hex output.
Recommend whether we should extend the test for one more week or call it now.
Keep the analytics as evidence, not as a separate analysis workflow.
```

## Useful Output Shape

- Recommendation
- Decision needed
- Options
- Trade-offs
- Evidence
- Risks
- Reversal cost
- Next step
