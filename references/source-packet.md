# Source Packet

Any skill in this repo may optionally use this compact source block when the user wants connected or pasted context grounded in a bounded window.

```text
SOURCE PACKET
Sources checked:
- Slack:
- Gmail:
- Calendar:
- Zoom transcript:
- Atlassian:
- Hex / data output:

Source gaps:
- ...

Confidence:
- High:
- Medium:
- Low:
```

Guidance:

- Read before write.
- Prefer small windows like "since yesterday", "this week", "since the last meeting", or "this thread".
- Separate source-backed facts from assumptions and suggestions.
- Surface missing or stale inputs.
- Summarize into decisions, actions, risks, asks, and follow-ups instead of dumping raw source content.
